godaddy deploymnet location (dep_loc): ~/public_html/mansoob-ui/

in ~/public_html/: make sure .htaccess file:

RewriteEngine On
DirectoryIndex disabled
RewriteRule ^$ http://127.0.0.1:3000/ [P,L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ http://127.0.0.1:3000/$1 [P,L]
---------

1- workstation run: ng build --prod
2- scp/ftp dist/* to dep_lob/dist/*
3- mv/clone(git) nodejs backend
4- dep_loc run: npm i pm2 -g
5- pm2 start server.js --watch or indirectory pm2 start run.sh

MySql connection configuration:
Host:localhost, port:3306, user:manssobdb, db:mansoob
